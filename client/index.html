Unlimited Skies

Unlimited Skies is a free browser game / passion project by Adrian Margel.

Remember when games were about having fun? This game is inspired by the Flash games of old. No drawn-out gameplay, no microtransactions. Just a cool idea taken to its limit. Unlimited Skies is a 2D plane dogfight game with a crazy selection of planes which are all infinitely upgradable. Play against increasingly insane waves of enemies with increasingly insane planes, until your computer can't handle any more.

If you're bored or looking for something new, why not give this free shooter game a try?

This game is playable on mobile. However, it may be laggy on some devices.

If the game does not run, make sure you are running the latest version of Firefox or Chrome.
